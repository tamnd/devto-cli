First release. The binary is `devto`.

## What it does

Reads public DEV Community data through `https://dev.to/api` with no API key
or account. Four commands:

- `devto top [--period week|month|year|alltime] [-n N]` — top articles over a
  chosen time window. Default period is the past week.
- `devto tag <tag> [-n N]` — most recent articles tagged with a DEV tag slug
  (e.g. `go`, `python`, `webdev`).
- `devto user <username> [-n N]` — user profile followed by that user's articles.
- `devto article <id>` — single article by integer id.

All commands share the usual output flags: `-o table|json|jsonl|csv|tsv|url|raw`,
`--fields`, `--no-header`, `--template`, `-n`, `-q`.

## Installation

Download a binary from the release assets below, or install via Homebrew:

```
brew install tamnd/tap/devto
```

Or with Go:

```
go install github.com/tamnd/devto-cli/cmd/devto@latest
```

## Quick start

```
devto top -n 10
devto top --period month -n 5 -o json
devto tag go -n 20
devto tag javascript -o jsonl | jq .url
devto user ben -n 5
devto article 12345
```
